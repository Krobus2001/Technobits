# 📖 About

TECHNOBITS is a student-led technology initiative that provides affordable and reliable technical services within the school community.

This website serves as the official digital platform where students can:

- 💻 View available technical services
- 📅 Stay updated on upcoming events
- 🏆 Register for tournaments and activities
- 📢 Receive club announcements
- 📞 Contact the TECHNOBITS team

The goal is to create a clean, responsive, and user-friendly experience while showcasing the club's professionalism.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/yourusername/TECHNOBITS.git
```

## Navigate into the project

```bash
cd TECHNOBITS
```

## Install dependencies

```bash
npm install
```

## Configure Environment Variables

Create a `.env.local` file.

```env
NEXT_PUBLIC_SUPABASE_URL=your_project_url

NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```

> **Important**
>
> Never commit your `.env.local` file or any secret keys to GitHub.

---

## Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
